Enclosed are two seperate projects:
--------------
Performance Counters - illustrating simple usage of performance counters
to monitor your application.
-------------
Trace Listeners - Illustrates how to use the Enterprise Library Logging 
Application block to log message from your application to various locations.
Right now I just show you XML, Plain Text, Event Log and Plain Text 
rolling file (New one every so often).
-------------
To run either of these projects...right click on the project 
and choose Set as Startup Project.  F5 to run.
